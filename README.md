# Jenkins-test, a demo Jenkinsfile for developer

Jenkins-test, a demo Jenkinsfile for developer

You can find some useful Jenkins Pipeline usage here.

## Getting started

1. Clone the repo

2. Install pre-commit

`$ pip install pre-commit`

3. Install pre-commit hooks

`$ pre-commit install`
